# Производственная практика

## Описание
## Программный модуль «Каталог личной онлайн-библиотеки»

автоматизация процесса учёта домашней библиотеки, предоставление удобного интерфейса для добавления, редактирования и поиска книг, а также отслеживания прогресса чтения.

## Содержимое репозитория

| Файл | Описание |
|------|----------|
| `Техническое задание.docx` | Требования к модулю по ГОСТ 19.201-78 |
| `Анализ Предметной Области.docx` | Сущности, бизнес-правила, роли пользователей |
| `Диаграмма классов, коммуникаций, компонентов, последовательностей, развёртывания, состояний` | UML-диаграммы (Draw.io) |
| `LibraryEXE` | Приложение WPF (.exe) |
| `LibraryApp` | Приложение WPF (Visual Studio) |
| `README.md` | Описание проекта |

## Инструкция по работе

1. Клонировать репозиторий
2. Открыть файлы в соответствующих программах:
   - `.docx` — Microsoft Word
   - `.drawio` — Draw.io
   - `.csproj` — Visual Studio
